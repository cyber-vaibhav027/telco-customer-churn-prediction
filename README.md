# Telco Customer Churn Prediction

## Project Overview
This project focuses on building a predictive classification model to identify which telecom customers are highly likely to churn. This empowers businesses to deploy proactive retention strategies.

## Task 1: EDA & Data Cleaning Key Findings
* **Class Imbalance:** ~26.5% of the customer base has churned.
* **Contract Risks:** Month-to-month contract accounts display the highest churn rates.
* **Tenure Impact:** Most churning customers leave within their first 12 months.

## Tools Used
